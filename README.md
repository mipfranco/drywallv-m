drywallv-m

Aplicação para gerenciamento de projetos de drywall.

🌐 Acesse

👉 https://mipfranco.github.io/drywallv-m/

💡 Sobre

Projeto desenvolvido para organizar e gerenciar informações de forma simples e eficiente, servindo também como prática de desenvolvimento web.

🛠️ Tecnologias
JavaScript
Node.js
HTML & CSS
⚙️ Como rodar
git clone https://github.com/mipfranco/drywallv-m.git
cd drywallv-m
npm install
npm run dev
🚀 Deploy
